# 学习通答案批量提交工具

从学习通作业页面自动解析题目（ID、类型、文本、选项），匹配答案后一键 POST 提交。

## 功能

- 🔍 **页面解析** — 粘贴作业 URL + Cookie，自动提取 token、题目 ID、题目文本、选项
- ✏️ **答案预览** — 输入答案后实时预览，自动识别单选/多选/判断题
- 📋 **题目对照** — 解析后显示每题的题干和选项，方便核对答案
- 🚀 **一键提交** — 使用页面提取的实际题目 ID（非连续），确保答案正确落位
- 📂 **文件加载** — 支持从 `.md` / `.txt` 文件加载答案列表

## 快速开始

### 安装

```bash
pip install -r requirements.txt
```

### 启动 Web 工具

```bash
python run.py
```

浏览器自动打开 `http://127.0.0.1:5000`。

### 使用步骤

1. 打开学习通作业页面，F12 → Network → 复制 Cookie
2. 复制浏览器地址栏的作业页面 URL（`dowork?courseId=...`）
3. 粘贴到工具的 **请求参数** 区域，点击 **🔍 解析**
4. 解析成功后，在右侧 **答案** 区域逐行输入答案（或点击 📂 加载文件）
5. 下方 **答案解析预览** 自动显示题目、选项与答案的对照表
6. 核对无误后点击 **🚀 提交答案**

## 项目结构

```
├── run.py                  # Flask Web 服务入口
├── requirements.txt        # 依赖
├── chaoxing/
│   ├── core.py             # 核心提交逻辑（POST 请求构建与发送）
│   ├── chaoxing_post.py    # CLI 命令行版本
│   ├── templates/
│   │   └── index.html      # Web 前端界面
│   └── 正确答案.md          # 默认答案文件（示例）
└── debug/                  # 调试文件（已 gitignore）
```

## CLI 使用

```bash
python chaoxing/chaoxing_post.py
```

按提示输入：请求网址、Cookie、题目 ID 列表（逗号分隔）、答案文件路径等。

> 推荐使用 Web 版，自动解析页面省去手动填写。

## 注意事项

- Cookie 有时效性，过期需重新获取
- 题目 ID 必须从页面提取（不连续），**不能**用首题 ID 推算
- `学习通post.md` 含敏感信息，已加入 `.gitignore`
